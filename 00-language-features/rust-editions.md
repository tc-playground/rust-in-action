# Rust Editions

## Introduction

* There are two `editions` of Rust:

    1. `2015 Edition`

    2. `2018 Edition`

* The edition can be specified by a flag when compiling, e.g. `rustc --edition=2018`

* The edition can be specified in the `Cargo.toml` configuration file:

    ```toml
    [package]
    edition = "2018"
    ```
